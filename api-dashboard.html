Vagaro

Enterprise partner-gated
Base: https://api.vagaro.com/{region}/api/v2 · docs
Auth: OAuth2 client-credentials — POST /merchants/generate-access-token with {clientId, clientSecretKey, scope}. Token passed as accessToken: header (not Bearer).
Access: Paying merchants only. Credentials via Vagaro Enterprise Sales. $10/mo + 5,000 calls included + $0.002/call beyond.

GlossGenius

No public API
Private backend only · Consumer booking site: glossgenius.com/{slug}/booking
Zero official API. All discovery paths return 404. GG has no SDK, no developer portal, no Zapier, no partner program. Industry-confirmed.

Private API (reverse-engineered — ToS risk)

Host api.glossgenius.com is live (Go-style gateway). Consumer booking flow at glossgenius.com/<slug>/booking is Next.js client-rendered, unauthenticated per slug.
GET/v1/public/businesses/{slug}Business profile (inferred)
GET/v1/public/businesses/{slug}/servicesService catalog
GET/v1/public/businesses/{slug}/staffStaff list
GET/v1/public/availabilitySlot lookup
POST/v1/public/bookingsCreate booking (needs Stripe PM token)

Sanctioned backdoor

Google Calendar sync — if a client grants Calendar OAuth, booked GG appointments appear as GCal events. Only officially-sanctioned read path.
Zoca strategy: Direct bookings through the public glossgenius.com/<slug>/booking URL (captured in ~/glossgenius_booking_links.csv). For automated booking, capture the network trace in Playwright and script the private endpoints — treating them as an undocumented contract that can break anytime.

Fresha

No public API
